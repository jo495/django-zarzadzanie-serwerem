Prototyp aplikacji do zarządzania domowym serwerem, mający docelowo pozwalać poszczególnym użytkownikom włączać kontenery z aplikacjami na serwerze, żeby nie zużywały zasobów przez cały czas.
Po utworzeniu usera, admin musi mu przydzielić grupę(ExtUsers, IntUsers) i dodać obiekt customer, któremu nadaje konkretnego usera oraz jego poziom dostępu.
Poziom dostępu decyduje, których obiektów Button będzie mógł użyć.
